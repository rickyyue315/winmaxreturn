## 📋 使用流程

1. **啟動應用**: `streamlit run app.py`
2. **上傳數據**: 選擇 Excel 文件或使用默認文件
3. **預覽數據**: 檢查數據完整性和格式
4. **生成建議**: 點擊分析按鈕開始處理
5. **查看結果**: 瀏覽退貨建議和統計圖表
6. **質量檢查**: 確認所有驗證項目通過
7. **下載報告**: 獲取完整的 Excel 分析報告
